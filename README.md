# Carta Digital - Backend

Este proyecto consiste en una carta digital para restaurantes en Django y React. Entre otras funciones, los usuarios pueden escoger una mesa, seleccionar platillos y agregarlos a un carrito, solicitar una orden pedido y pedir la cuenta para pagar, ya sea en efectivo o con tarjeta de cr√©dito.

## Clonar Repositorio

    git clone https://github.com/aranibar28/back-card.git

## Flujo de Trabajo

    python -m venv env    
    . env/Scripts/activate
    pip install -r requirements.txt
    python manage.py runserver
